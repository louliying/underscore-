原生ES5中的some

属于Array.prototype.some()

判断数组，是否有一个元素，是满足迭代器的要求，即迭代器返回true的。如果有一个满足，中止运行迭代器，返回true。

        array.some(function(currentValue,index,arr),thisValue)
        // currentValue, 必须，当前元素的值
        // index  可选。当前元素下标
        // arr  可选。被迭代的数组对象
        // thisValue    可选。对象作为该执行回调时使用，传递给函数，用作 "this" 的值。如果省略了 thisValue ，"this" 的值为 "undefined"

some不会对空数组遍历， 不会改变 原始数组。

         var arr = [1, 2, 3];
         var res = arr.some(function(value){
            return value % 2 == 0
        });
        console.log(res);
